### 2. 推理過程 (N+1 / N+2)
基於講師提出的 CT 原則，我們可以推論在 IELTS Writing Task 2 的具體套用步驟：

1.  **步驟一：拒絕直覺反應（對抗「懶惰思考」）**
    *   *邏輯：* 當看到題目（如：政府應禁售菸酒）時，大腦會產生直覺偏見。
    *   *做法：* 批判性思考者會先問「這是真的嗎？禁令真的有效嗎？」而不是直接開始寫贊成票。這能幫你找到更深層的論點（例如：黑市問題），提高 Task Response 的分數。

2.  **步驟二：構建邏輯鏈條（對抗「重複論點」）**
    *   *邏輯：* 講師提到不要只說「我是對的」，要提供「證據與推理過程」。
    *   *做法：* 在 Body Paragraph 中，不能只寫「暴力遊戲有害（論點）」。必須展示邏輯：`暴力遊戲 -> 脫離現實的暴力行為模擬 -> 長期暴露導致對暴力麻木 -> 可能增加侵略性行為`。這就是 CT 中的「思維過程透明化」。

3.  **步驟三：客觀評估對立面（對抗「確認偏誤」）**
    *   *邏輯：* 講師強調要「看見兩面性」(00:16:42) 且「評估不同論點的強度」。
    *   *做法：* 在「Discuss both views」題型中，不能厚此薄彼。應客觀分析：`「支持方認為 A 是基於 B 理由，雖然這在 C 情況下成立，但考慮到 D 證據，其影響力有限。」` 這種寫法展現了講師所說的「理性評估」。

4.  **步驟四：自我反省論證結構（套用「自我覺察」）**
    *   *邏輯：* 檢查自己的論證是否受困於「文化或教育偏誤」。
    *   *做法：* 寫完後自問：`「我這個論點是因為『大家都這麼說』還是『邏輯上真的成立』？」` 避免使用過於絕對的詞彙（如 All, Always），改用更具批判性的限制詞（如 tend to, in certain circumstances）。
