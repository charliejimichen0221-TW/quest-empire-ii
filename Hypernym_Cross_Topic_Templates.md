## §1 九大 Hypernym 家族 + 萬用句型

> 每個家族包含一組高頻替換詞和 3 條萬用句型。寫作時判斷題目屬哪個家族 → 直接套。

### 🏛️ 1. 制度與政策

| Hypernym | 核心替換詞 | 延伸同義 |
|---|---|---|
| {政策制定者} | the government / authorities / policymakers | decision-makers / governments |
| {法規} | regulation / legislation / law / policy | restrictions / reform / legalisation |
| {公共資金} | public funds / government budget / subsidies | taxpayer money / grants / funding / donations |
| {公共服務} | public services / healthcare / education | public safety / infrastructure / public institutions |
| {獨立監管} | independent oversight / regulators | ethics committees / accountability / transparency |
| {國際合作} | international cooperation / diplomacy | foreign aid / global agreements / wealthy nations |
| {禁令} | ban / prohibition / restriction | censorship / crackdown |

- The government must **regulate {X}** and fund {Y} to protect {受影響群體}.
- Rather than {極端 A or B}, a balanced approach would be to **{折衷方案}**.
- {法規} should **require {transparency / accountability}** while protecting {freedom / innovation}.

### 👤 2. 受影響群體

|---|---|---|

| {勞工} | workers / employees / professionals | low-skilled workers / workforce |
| {弱勢族群} | the elderly / low-income families / minorities | the disabled / vulnerable people / single parents |
| {納稅人} | taxpayers / citizens / the public | residents / individuals / society |
| {家庭} | families / parents / children / guardians | a parent / household |

| {難民/移民} | refugees / immigrants / migrant workers | asylum seekers / displaced people |
| {女性} | women / working mothers / gender equality | female empowerment |

- {群體} who {弱勢條件} deserve **{支援/保護}**, not {neglect/punishment}.

## §2 十二大萬用引擎（公式 + 範例）

> 每個引擎 = 1 個句型公式 + 5 個跨主題填入範例。

### ✅ E1「改善生活」
> **{X} makes {people's} life more {adj}** because they can {benefit}.

| 範例 |
|---|
| Art makes {individuals'} life more **peaceful** because they can escape {daily pressure} through {creative activities}. |
| Reading makes {young learners'} life more **intellectually enriched** because they can build {language skills}. |
| {Work-life balance} makes {workers'} life more **productive** because {well-rested workers} produce {better work}. |
| E-books make {travellers'} life more **accessible** because they can carry {an entire library} on {a single device}. |
| {Studying abroad} makes {students'} life more **globally competitive** because they gain {cross-cultural skills}. |

### ❌ E2「造成傷害」
> **It damages {group's} {aspect}** when {cause}, leading to **{negative consequence}**.

| 範例 |
|---|
| It damages **{mental health}** when {teenagers} compare themselves to {filtered, fake online lives}. |
| It damages **{employment}** when {AI} replaces {millions of jobs} in {manufacturing}. |
| It damages **{public services}** when {rapid population growth} outpaces {investment}. |
| It damages **{sleep}** because {blue light and notifications} disrupt {natural sleep patterns}. |
| It damages **{animal welfare}** when {sentient beings} are {tested on} for {human convenience}. |

### ❌ E3「太貴了」
> **It is too expensive** for {group} when {cost situation}, resulting in **{negative outcome}**.

| 範例 |
|---|
| Too expensive when {billions} lack {clean water} while {a space agency spends billions}. |
| Too expensive for {most families} — {EVs} cost {significantly more} than {petrol cars}. |
| Too expensive — {host cities} typically {overspend by billions}, leaving {taxpayers} with {debt}. |
| Too expensive when {building codes} increase {construction costs} and {delay housing supply}. |
| Too expensive when a {country's citizens} lack {housing} while {billions are sent overseas}. |

### ⏩ E4「如果…就會…」（正向）
> **If {positive action}, {beneficiary} will {positive result}.**

| 範例 |
|---|
| If {gender equality} is achieved, **{economies and societies} all benefit**. |
| If proven safe, {driverless cars} will **save {millions of lives}**. |
| If {privacy} is protected, **{citizens will trust digital services more}**. |
| If reading is encouraged, children develop **stronger {linguistic ability}**. |
| If managed, {tourism} will **benefit {local economies}** without {destroying nature}. |

### ⏩ E5「如果不…就會…」（負面）
> **Without {action}, we face {dire consequence}.**

| 範例 |
|---|
| Without {action}, we face **{a post-truth society}** where {facts} no longer matter. |
| Without {reducing emissions}, we face **{climate catastrophe}**. |
| Without {retraining}, we face **{mass unemployment}** and {growing inequality}. |
| Without art funding, we face **a {spiritually unfulfilled} society**. |
| Without {reform}, we face **{a permanent renting class}** with no {security}. |

### 👁️ E6「我們經常看到」
> **We often see {observable phenomenon}** — {context or irony}.

| 範例 |
|---|
| We often see **{universities requiring volunteer hours}**, turning {altruism} into {strategy}. |
| We often see **{influencers promoting products without disclosing payment}**. |
| We often see **{elderly parents left in care homes}**, visited only on {special occasions}. |
| We often see **{extreme sport videos going viral}**, encouraging {untrained people to try stunts}. |
| We often see **{employees responding to emails at midnight}** — {work-life boundary dissolved}. |

### 👁️ E7「統計顯示」
> **Statistics show {data point}** — {implication}.

| 範例 |
|---|
| Statistics show **{billions use social media}** — more than {half the global population}. |
| Statistics show **{the richest fraction own nearly half of global wealth}**. |
| Statistics show **{nuclear provides a significant share of electricity}** but is declining. |
| Statistics show **{billions play video games}** — the biggest {entertainment industry}. |
| Statistics show **only {a small proportion of parliamentarians}** are {women}. |

### 🔧 E8「政府必須」
> **The government must {policy action}** and {complement} to {outcome}.

| 範例 |
|---|
| Must **enforce {data protection laws}** and give {citizens} the right to {delete data}. |
| Must **regulate {AI}** and fund {retraining} for {displaced workers}. |
| Must **build {more homes}, regulate {rents}**, and help {first-time buyers}. |
| Must **enforce {maximum working hours}** and give {workers} the right to {disconnect}. |
| Must **enforce {building safety standards through independent inspectors}**. |

### 🔧 E9「學校應該」
> **Schools should teach {skill}** so {students} can {application}.

| 範例 |
|---|
| Teach **{media literacy}** so {students} can recognise {persuasion techniques}. |
| Teach **{financial literacy}**: {compound interest, credit ratings, debt cost}. |
| Teach **{digital literacy}** so {students} can identify {fake news}. |
| Teach **{environmental science}** so children understand {their carbon footprint}. |
| Teach **{critical thinking about public figures}** so {students} don't blindly copy {lifestyles}. |

### 🔧 E10「折衷平衡」
> **Rather than {extreme A or B}, a balanced approach would be to {middle ground}.**

| 範例 |
|---|
| Rather than {banning nuclear}, **use {nuclear alongside renewables}**. |
| Rather than {open/closed borders}, **welcome {immigrants} while investing in {integration}**. |
| Rather than {banning all cloning}, **allow {medical applications} under {ethical oversight}**. |
| Rather than {always giving cash}, **combine {a meaningful gift with financial support}**. |
| Rather than {mandating vegetarianism}, **{encourage reduced meat with nutritional education}**. |

### 🔄 E11「反方觀點」
> **The opposing view argues that {counter-argument}, but this overlooks {rebuttal}.**

| 範例 |
|---|
| The opposing view argues that {banning cars is impractical}, but this overlooks **{the success of car-free zones in European cities}**. |
| The opposing view argues that {competition motivates students}, but this overlooks **{the mental health cost of constant pressure}**. |
| The opposing view argues that {fast food is a personal choice}, but this overlooks **{children's inability to make informed decisions}**. |
| The opposing view argues that {refugees burden the economy}, but this overlooks **{their long-term tax contributions and workforce value}**. |
| The opposing view argues that {AI creates more jobs}, but this overlooks **{the skills mismatch facing displaced workers}**. |

### 📌 E12「具體舉例」
> **For instance, in {country/context}, {specific example}, which demonstrates {point}.**

| 範例 |
|---|
| For instance, in **{Australia}**, {banning smoking in restaurants reduced waiter lung cancer rates}, demonstrating **{regulation saves lives}**. |
| For instance, in **{Finland}**, {students rarely take high-stakes exams yet rank top globally}, demonstrating **{less pressure can produce excellent results}**. |
| For instance, **{Steve Jobs}**, {the son of a Syrian immigrant}, demonstrates **{refugees can contribute enormously to a host nation}**. |
| For instance, in **{Japan}**, {schools require students to join a sports club}, demonstrating **{structured activity prevents childhood obesity}**. |
| For instance, **{Tesla's electric vehicles}**, {which have zero tailpipe emissions}, demonstrate **{technology can solve environmental problems}**. |

## §4 即時寫作流程

```
Step 1 → 看題目 → 查 §3 找家族
Step 2 → 查 §1 拿 hypernym 替換詞
Step 3 → 查 §2 選 2-3 個引擎
Step 4 → 填入 hypernym → 寫完一個段落！
```

> 💡 **選好家族 + 引擎後** → 對照 [Cluster × Hypernym 橋接表](Cluster_Hypernym_Bridge.md) 可看具體集群的配方范例！

### 💡 實戰示範

**題目：** *"The fashion industry is bad for the environment. Do you agree?"*

**Step 1：** Exam 13 → 🔄時尚產業 + 🌍環境 → E2, E5, E7

**Step 2-3 填入：**

✅ **E1：** {The fashion industry} makes **{workers'} life more {economically secure}** because it creates {millions of jobs}.

❌ **E2：** It damages **{the environment}** when {fast fashion} encourages {overconsumption and textile waste}, leading to **{massive landfill pollution}**.

⏩ **E5：** Without {sustainable practices}, we face **{environmental catastrophe}** from {billions of discarded garments annually}.
